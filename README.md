# Riding Cities

Riding Cities est un mini-site vitrine consacre a une association de skateboard. Le projet sert d'introduction aux bases HTML/CSS (structure, mise en page et liens de telechargement).

## Apercu

- Page unique responsive avec hero, sections de mission et equipe.
- Mise en page en colonnes pour les blocs d'information.
- Deux liens de telechargement (plannings cours adultes/enfants).

## Contenu

- Presentation de l'association et de sa mission.
- Portraits des membres fondateurs.
- Section de telechargement des plannings de cours.

## Stack technique

- HTML5
- CSS3

## Lancer le projet

Ouvrir `index.html` dans un navigateur.
